Ínicio da internet

(Voltar)

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto")

é uma linguagem de marcação utilizada na construção de páginas na Web.

Em 1991 no CERN(European Council for Nuclear Research) na suíça o HTML foi criado por Tim Berners-Lee onde inicialmente

foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade.

Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web),

uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

Como funciona o HTML

(Voltar)

O HTML é uma linguagem de marcação. estas linguagens são constituídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria

onde possuem códigos para criar paginas na web.

códigos html podem definir o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site.

O HTML foi a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web,

porém o HTML ainda prevalece. Atualmente já é possível integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site.

Para criar e editar códigos em HTML é necessário qualquer editor de texto comum, como bloco de notas. Para testar os códigos,

basta salvar o arquivo em formato .HTML e executar. Para o teste é necessário ter um navegador configurado como padrão.

Não é necessária internet, pois o arquivo com os códigos esta na maquina onde esta sendo executado.
